
# Documentação do Projeto

Este repositório contém os arquivos necessários para o funcionamento do projeto,
organizados em diferentes pastas para facilitar o entendimento e uso.

## Estrutura das Pastas

### 1. Pasta `projeto`
- Contém:
  - **Código Arduino:** Arquivo com o programa utilizado para o microcontrolador.
  - **Programa do Elipse E3:** Arquivo do projeto desenvolvido no Elipse E3 para supervisão e controle.

### 2. Pasta `bibliotecas`
- Contém as bibliotecas utilizadas no projeto:
  - **Biblioteca Modbus (Arduino):** Necessária para implementar a comunicação Modbus no Arduino.
    - Inclui um exemplo que demonstra a declaração de todos os registradores das respectivas portas do controlador.
  - **Biblioteca do Elipse E3:** Fornece suporte à comunicação com o software Elipse E3.
